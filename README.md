# WarnaBantu
WarnaBantu AI adalah aplikasi Android untuk membantu penyandang buta warna mendeteksi warna dengan akurat. Fitur utamanya mencakup AI Color Recognition, pemindaian real-time bersuara (TTS), analisis kontras WCAG, dan database offline. Aplikasi ini dirancang ringan, hemat baterai, dan berdesain modern.
